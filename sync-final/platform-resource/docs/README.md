# platform-resource docs

`platform-resource` 문서 기준은 다음과 같다.

## 범위

- 파일/오브젝트 저장 추상화
- 알림 전달 추상화
- 저장소 경로와 ACL 규칙
- delivery adapter 조합
- Spring 연동
- API 오류 모델

## 범위 밖

- 인증
- 정책 평가
- 감사 저장
- 서비스 비즈니스 로직
