# oss-contract

- 1계층 OSS는 순수 기능 모듈이다.
- 2계층 platform은 소비 애플리케이션이 붙는 조립 계층이다.

## 디렉터리 역할

- `registry`: 계층 기준, 운영 기준, 문서 / build / publish 구조 기준을 담는다.
- `repositories`: 실제 레포 카탈로그, 최신 기준 버전, 레포별 현재 상태와 정리 후보를 담는다.
