# 다음 작업

이 문서는 지금 상태에서 바로 이어서 할 작업을 적는다.

## 우선순위

1. `auth` 실제 레포에 `sync-final/auth` 내용을 반영한다.
2. `service-contract` 실제 레포에 `sync-final/service-contract` 내용을 반영한다.
3. `auth`와 `service-contract`의 용어 차이를 제거한다.
4. 나머지 OSS 레포를 같은 방식으로 반영한다.

## 작업 방식

- 먼저 로컬 초안을 비교한다.
- 쓰기 권한이 있으면 바로 sync한다.
- 쓰기 권한이 없으면 patch만 준비한다.
